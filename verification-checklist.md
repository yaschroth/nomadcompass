# The Nomad HQ Verification Checklist

---

## Tenant (Digital Nomad) Verification

### Required (Must Pass All)

- [ ] **Email verified** — Clicked confirmation link
- [ ] **Phone verified** — SMS code confirmed
- [ ] **ID document uploaded** — Passport or national ID (clear photo, not expired)
- [ ] **Selfie matches ID** — Face matches document photo
- [ ] **Video call completed** — 5-10 min intro call, face matches ID

### Strongly Recommended

- [ ] **LinkedIn profile linked** — Active profile, employment history visible
- [ ] **Income verification** — One of:
  - [ ] Recent bank statement (showing regular income)
  - [ ] Pay stub (last 3 months)
  - [ ] Employment contract
  - [ ] Letter from employer
  - [ ] Proof of freelance income (invoices, client contracts)
- [ ] **Previous landlord reference** — Contact info provided and verified
- [ ] **Social media check** — At least one active profile (LinkedIn, Twitter, Instagram)

### After First Stay

- [ ] **Landlord review received** — Rating + written feedback
- [ ] **Payment completed successfully** — No chargebacks or issues
- [ ] **Trusted Tenant badge** — Awarded after 1+ successful stays

### Red Flags — Reject or Investigate Further

- [ ] Won't do video call (any excuse)
- [ ] ID looks altered or low quality
- [ ] Selfie doesn't match ID
- [ ] No online presence whatsoever
- [ ] Vague about work ("I do business online")
- [ ] Wants to pay only in crypto/cash/wire transfer
- [ ] Pressures for immediate booking without verification
- [ ] Email domain looks suspicious (random strings)
- [ ] Phone number is VOIP/temporary

## Landlord (Property Owner) Verification

### Required (Must Pass All)

- [ ] **Email verified** — Clicked confirmation link
- [ ] **Phone verified** — SMS/WhatsApp confirmed
- [ ] **Property photos submitted** — Minimum 8 photos covering:
  - [ ] Exterior/building entrance
  - [ ] Living area
  - [ ] Bedroom(s)
  - [ ] Kitchen
  - [ ] Bathroom
  - [ ] Workspace/desk area
  - [ ] View from window
  - [ ] WiFi router location
- [ ] **Live video tour completed** — 10-15 min walkthrough via WhatsApp/Zoom
  - [ ] Matches submitted photos
  - [ ] Shows all rooms
  - [ ] Demonstrates WiFi speed test (live)
  - [ ] Shows view from window (confirms location)
  - [ ] Shows building entrance/street (matches Google Maps)

### Strongly Recommended

- [ ] **Proof of property control** — One of:
  - [ ] Utility bill (with property address + their name)
  - [ ] Property ownership document
  - [ ] Property management contract
  - [ ] Rental license (where required)
- [ ] **Google Maps verification** — Street view matches:
  - [ ] Building exterior
  - [ ] Neighborhood shown in photos
- [ ] **Cross-reference check** — Search for property on:
  - [ ] Airbnb
  - [ ] Booking.com
  - [ ] Local rental sites
  - [ ] Reverse image search photos
- [ ] **Reviews on other platforms** — If listed elsewhere, check ratings

### Property Details Confirmed

- [ ] **Address verified** — Full address provided and confirmed via video
- [ ] **WiFi speed tested** — Live speedtest.net during video call
  - Actual speed: ______ Mbps
- [ ] **Workspace confirmed** — Desk and chair visible
- [ ] **All-in price confirmed** — Monthly total includes:
  - [ ] Rent: €______
  - [ ] Utilities: €______ (or included)
  - [ ] WiFi: €______ (or included)
  - [ ] Cleaning: €______ (or included)
  - [ ] **Total all-in: €______**
- [ ] **Deposit amount** — €______ (max 1-2 months)
- [ ] **Minimum stay** — ______ days/months
- [ ] **Cancellation policy** — Documented and agreed

### After First Tenant

- [ ] **Tenant review received** — Rating + written feedback
- [ ] **Property matched listing** — No complaints about accuracy
- [ ] **Verified Host badge** — Awarded after 1+ successful stays

### Red Flags — Reject or Investigate Further

- [ ] Won't do live video tour (any excuse)
- [ ] Photos look like stock images or too professional
- [ ] Reverse image search shows photos on multiple unrelated listings
- [ ] Price significantly below market rate (too good to be true)
- [ ] Requests payment before signing contract
- [ ] Only accepts wire transfer/Western Union/crypto
- [ ] Claims to be "abroad" and can't show property
- [ ] Doesn't know basic details (which floor, nearest metro, etc.)
- [ ] Pressures for quick payment/deposit
- [ ] No proof of property control
- [ ] Address doesn't exist or doesn't match Google Maps
- [ ] Communication only via email (won't do phone/video)

## Payment Safety Rules

### Never Allow

- [ ] Direct bank transfers between tenant and landlord before check-in
- [ ] Cash payments (no paper trail)
- [ ] Crypto payments (irreversible)
- [ ] Western Union / MoneyGram (scammer favorite)
- [ ] Payment before contract is signed
- [ ] Payment to different name than landlord

### Always Require

- [ ] Payment through platform (Stripe, escrow)
- [ ] Signed rental agreement before any payment
- [ ] Deposit held in escrow until check-in confirmed
- [ ] 24-48 hour check-in confirmation window
- [ ] Clear refund policy if property doesn't match

## Verification Call Scripts

### Tenant Video Call (5-10 min)

```
1. "Hi [Name], thanks for joining! Can you hold up your ID next to your face for a moment?"
   → Confirm match

2. "Tell me a bit about what you do for work."
   → Verify matches LinkedIn/application

3. "Where are you currently based, and why are you looking to stay in [City]?"
   → Check for consistency, legitimate reason

4. "Have you done monthly rentals before? Any references?"
   → Note for follow-up

5. "Any questions about how The Nomad HQ works?"
   → Explain payment protection, verification
```

### Landlord Video Call (10-15 min)

```
1. "Thanks for your interest in listing with us. Can you show me around the property?"
   → Walk through all rooms, match to photos

2. "Can you show me the workspace area?"
   → Confirm desk, chair, lighting

3. "Let's do a quick WiFi test — can you go to speedtest.net?"
   → Record actual speed

4. "Can you show me the view from the window / building entrance?"
   → Confirm location matches address

5. "What's the total monthly cost including everything?"
   → Document all-in price

6. "What's your cancellation policy?"
   → Document terms

7. "Do you have a utility bill or document showing you manage this property?"
   → Request upload after call
```
